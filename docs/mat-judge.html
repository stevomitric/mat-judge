Execute, judge and evaluate code in an isolated enviroment

Judge ¶

Submission  ¶

Send code for evaluation and judgeing process. Access level 3 (user)

Submit
POST/submit

Example URI

POST http://localhost:5000/api/v0/submit
URI Parameters
HideShow
code
string (required) 

Code to be executed

language
string (required) 

Language short id

Choices: c++11 c++14 c++17 c11 py3

time_limit
number (optional) 

Time limit in miliseconds. Must be => 100 and <= 15000

  • default 1000
memory_limit
number (optional) 

Memory limit in MB Must be => 4 and <= 256

  • default 256
testcases
string (optional) 

List of tuples. Tuple values are strings. First value is input passed to the program, and second value is expected output.

testcases_id
number (optional) 

ID of stored testcases

grader
string (optional) 

Grader script to judge AC/WA based on programs & expected outputs. See Grader script information for more details and required format.

Request
HideShow
Headers
Content-Type: multipart/form-data
Body
{
        "code": '#include <iostream>
                 int main() {
                     std::cout << "Hello World!";
                     return 0;
                 }',
        "language": "c++11",
        "time_limit": 1000,
        "memory_limit": 1000,
        "testcases": [ ("1 2", "3"),  ("-10 7", "-3"), ("10 11", "21") ]
    }

    "success": true,

}

Status  ¶

Fetch status of a submission. Access level 4 (public)

Status
GET/status?submission_id={submission_id}

Example URI

GET http://localhost:5000/api/v0/status?submission_id=submission_id
URI Parameters
HideShow
submission_id
number (required) 

Submission identifier

output
bool (optional) 

Wheter to return output or not

  • default false
Request  /status?submission_id=154
Response  200
HideShow

If the submission has been fully judged. code Field can have following responses: AC (Accepted), WA (Wrong answer), RE (Runtime Error), CE (Compile Error), TLE (Time Limit Exceded), MLE (Memory Limit Exceded)

Headers
Content-Type: application/json
Body
{
    "success": true,
    "code": "AC",
    'run_time': 250,
    'running': false,
    'judged': 100,
}
Response  200
HideShow

If the submission runs into CE, compiler filed will be provided

Headers
Content-Type: application/json
Body
{
  "success": true,
  "code": "CE",
  "running": false,
  "Compiler": "Undefined xxx"
}
Response  200
HideShow

If the submission is running, judged filed will point out how many testcases have been judged so far

Headers
Content-Type: application/json
Body
{
    "success": true,
    "running": true,
    "judged": 24,
}
Response  200
HideShow
